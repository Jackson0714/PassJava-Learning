**这些对象和 Redis 交互时，有不同的操作：**

`客户端对象`会有网络 IO交互、键值对 增删改查 操作、数据库操作。

`磁盘对象`会有生成 RDB 快照、记录 AOF 日志、AOF 日志重写操作。

`主从节点`对象会有主库生成、传输 RDB 文件、从库接受 RDB 文件、清空数据库、加载 RDB 文件操作。。

`切片集群实例`对象会有向其他实例传输哈希槽信息、数据迁移操作。
