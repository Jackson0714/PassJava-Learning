# 程序员必备工具 1 - 抓包工具 - Fiddler

## Fiddler 能做什么

Fiddler 是一款强大的网络调试工具，主要用于捕获、分析和修改 HTTP 流量。

- 与MacOS，Windows和Linux兼容。

- 检查网络流量（HTTP和HTTps）。

- 解密安全流量（HTTps）。

- 与合作者保存，共享和接收捕获的流量。

- 模拟请求和响应

- 修改回应

- 撰写API请求

- 保存编写的API请求
- 等等
