## PassJava 项目简介

PassJava 是一款帮助Java面试的开源系统，采用流行的技术，如 SpringBoot、MyBatis、Redis、 MySql、 MongoDB、 RabbitMQ、Elasticsearch，采用Docker容器化部署。
